#Day 1-Linux Commands

pwd         - shows current directory
ls          - lists and folders
mkdir       - creates a folder
cd          - change directory
cd..        - go back one level
cd ~        - go to home folder
cp          - copy a file
mv          - move or renamr a file
rm          - delete a file
echo        - write text to a file
cat         - read a file
grep        - search inside a file
nano        - edit a file in terminal

ls -la      - show all files with details
cat         - read a file
chmod       - change file permissions
git status  - check what changed in repo
